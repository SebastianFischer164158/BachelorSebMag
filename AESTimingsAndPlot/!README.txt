#AESTimingsAndPlot Folder
Denne folder indeholder en liste over de enkelte filer.

AESTiming.ino er en Arduino fil, som benytter funktionaliteterne fra .h filen. 
Der bliver i denne fil timet AES funktioner.

AESplots.py, er et python script til at plotte resultaterne fra tidstagningerne som blev lavet via. AESTiming.ino filen. 

1. AES.h

2. AESTiming.ino

3. AESplots.py
